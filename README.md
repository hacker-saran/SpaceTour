# Space Tour

Welcome to Space Tour, my very first web development project built back in the day with pure HTML5 and CSS3. It was based on a Frontend Mentor challenge, and it represents the exact moment I realized coding is awesome (and that centering divs is a full-time job).

## Live Demo

See it live on GitHub Pages:
https://hacker-saran.github.io/SpaceTour/

## The Big Disclaimer

Since this was my literal first attempt at building a website:
- It only works properly on desktop screens.
- There is zero mobile responsiveness. If you open this on a phone, the layout will look like it went through a black hole. Please view it on a larger monitor to appreciate the layout as originally intended.

## Repository Organization

I recently revamped this repository to make it look clean and professional on GitHub. Here is how it is organized:

- CSS: Stylesheets for the Home, Destination, Crew, and Technology pages.
- Html: Nested HTML pages for the different sections of the tour.
- Images: Planets, crew profiles, and spaceship graphic assets.
- index.html: The main entrypoint page (moved to the root for clean hosting).

## Technologies Used

- HTML5 (raw tags and structure)
- CSS3 (flexbox, custom borders, and transition effects)
- Google Fonts (Barlow Condensed, Bellefair, and Poppins)
